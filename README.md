# API docs

`GET` **categories**

- /api/1/
- /api/1/categories
- /api/1/categories/{id}

`GET` **books**

- /api/1/books
- /api/1/books/{id}

`GET` **characters**

- /api/1/characters (mandatory parameter: **?search**)
- /api/1/characters/all
- /api/1/characters/{id}

`GET` **spells** (data collection is in progress)

- /api/1/spells
- /api/1/spells/{id}
